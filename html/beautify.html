How the live workflow works

Click Open a folder and choose the folder with your code. Open the same folder in VS Code (use Open VS Code, then File, Open Folder in vscode.dev). Edit and save; this page re-formats the file automatically.

Paste or load code, open a folder, pick a language, then Beautify.

Re-indent supported for JSON, JavaScript, CSS, HTML/XML, BibTeX, and Java. Other languages are highlighted as-is. Libraries load on demand.
