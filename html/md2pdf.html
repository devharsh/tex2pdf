How the live workflow works

Click Open a folder and choose the folder that holds your Markdown. Open the same folder in VS Code (use Open VS Code, then File, Open Folder in vscode.dev). Edit and save; this page re-renders automatically.

Output

Research paper mode converts Markdown to LaTeX and compiles a tagged, vector-graphics PDF with the same engine as the editor. Add a YAML front matter block for title, author, abstract.
