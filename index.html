TeX Viewer Online in-browser

View your TeX documents in any browser, no software needed. Compiles locally with WebAssembly; nothing is uploaded.

1. Engine

Which one should I choose?

Not sure? Keep XeLaTeX (the default). In this in-browser engine it handles fonts most reliably.

XeLaTeXRecommended default. Uses vector fonts directly, supports system or OpenType fonts (fontspec) and full Unicode.
pdfLaTeXSlightly faster, but it can fail when a document needs a bitmap font it has to build on the fly, which is not possible here. If you get a font error, switch to XeLaTeX.

    Live edit with VS Code

    Live folder needs Chrome or Edge. You can still upload files above.

    How the live workflow works

    Click Open a folder and choose your project folder. Then open the same folder in VS Code (use the Open VS Code button, then File, Open Folder in vscode.dev).

    Edit and save in VS Code. This tool watches the folder and rebuilds the PDF automatically on each save.
