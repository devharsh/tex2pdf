\section{Introduction}
This document was compiled entirely in a web browser using a WebAssembly build of
pdfTeX, then rendered to the page with PDF.js. No server performed the compilation,
and the source file was never uploaded anywhere.

\section{A short list}
\begin{itemize}
  \item Pick an engine: pdfTeX or XeTeX.
  \item Drop in a \texttt{.tex} file, several files, or a \texttt{.zip} project.
  \item Compile, preview, and download the PDF.
\end{itemize}
